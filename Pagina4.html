Nombre del Proyecto: Plataforma Educativa Local

Objetivos

Crear una plataforma accesible para estudiantes y docentes que permita compartir recursos, realizar evaluaciones y facilitar la comunicación institucional.

  • Facilitar el acceso a contenido educativo.
  • Permitir el seguimiento del progreso estudiantil.
  • Ofrecer herramientas de evaluación y retroalimentación.

Tecnologías utilizadas

  • HTML5
  • CSS3 (externo, no incluido en este ejercicio)
  • PHP / Node.js (para backend)
  • Base de datos MySQL
